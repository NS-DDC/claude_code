# 🎉 Saju-MBTI v2.0.0 - 대규모 기능 개선

## 새로운 기능

### 🔐 인증 시스템
- Firebase 인증 통합
- 이메일/비밀번호 로그인
- Google OAuth 소셜 로그인
- 보호된 라우트 (히스토리, 설정)

### ☁️ 클라우드 동기화
- Firestore 실시간 데이터베이스
- 오프라인 지원
- 기기 간 자동 동기화
- localStorage → Firestore 자동 마이그레이션

### 👤 사용자 프로필
- 프로필 사진 업로드
- MBTI 타입 저장
- 생년월일 정보 관리
- 활동 통계

### 🔔 푸시 알림
- 매일 운세 알림 (시간 설정 가능)
- 로컬 알림 + 푸시 알림
- Firestore 설정 동기화

### 🌙 다크 모드
- 라이트/다크 테마 전환
- Firestore에 테마 설정 저장
- 부드러운 전환 애니메이션

### 🎯 온보딩
- 4단계 환영 화면
- MBTI 타입 선택
- 생년월일 입력
- 알림 권한 요청

### 📤 데이터 관리
- 데이터 JSON 내보내기
- 기록 삭제 (확인 모달)
- 계정 삭제 (안전 확인)

### 📱 공유 기능
- 운세 결과 공유
- MBTI 궁합 공유
- 소셜 미디어 공유

### ⚡ 성능 개선
- 코드 분할 및 지연 로딩
- 번들 크기 최적화
- 로딩 스켈레톤

### ✅ 테스트
- Jest + React Testing Library
- 128개 테스트 통과
- 핵심 로직 80% 이상 커버리지

## 기술 사양

- **APK 크기:** 6.9 MB
- **최소 Android 버전:** 5.0 (API 21)
- **TypeScript:** 100% 타입 안전
- **의존성:**
  - Firebase 12.10.0
  - Next.js 16.1.6
  - Capacitor 5.6.0
  - React 18.2.0

## 설치 방법

1. **APK 다운로드**
   - Release 페이지에서 `saju_mbti.apk` 다운로드

2. **보안 설정**
   - 설정 → 보안 → 알 수 없는 출처 허용

3. **앱 설치**
   - 다운로드한 APK 파일 실행
   - 설치 진행

4. **Firebase 설정 (선택사항)**
   - 개발자인 경우 `.env.local` 파일 설정
   - Firebase 프로젝트 생성 후 credentials 입력
